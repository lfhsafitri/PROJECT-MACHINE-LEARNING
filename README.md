# PROJECT-MACHINE-LEARNING
DESKRIPSI PROJECT
Lembaga keuangan seperti bank dan perusahaan pembiayaan perlu melakukan evaluasi terhadap calon peminjam sebelum memberikan pinjaman. Proses ini bertujuan untuk meminimalkan risiko kredit macet dan memastikan bahwa pinjaman diberikan kepada individu yang memiliki kemampuan untuk melakukan pembayaran kembali.

Seiring berkembangnya teknologi dan data science, proses evaluasi pinjaman dapat dilakukan secara otomatis menggunakan teknik machine learning. Dengan memanfaatkan data historis peminjam, model klasifikasi dapat dibangun untuk memprediksi apakah suatu pengajuan pinjaman akan disetujui atau ditolak. Dataset yang digunakan berasal dari Kaggle dan berisi informasi finansial, aset, serta skor kredit pemohon pinjaman.
